
# HƯỚNG DẪN XÓA CACHE TRÌNH DUYỆT

## Chrome/Edge:
1. Nhấn Ctrl+Shift+Delete
2. Chọn "Cached images and files"
3. Nhấn "Clear data"

## Firefox:
1. Nhấn Ctrl+Shift+Delete
2. Chọn "Cache"
3. Nhấn "OK"

## Hoặc cách nhanh hơn:
1. Mửa trang profile
2. Nhấn Ctrl+F5 (hoặc Ctrl+Shift+R) để force reload
3. Hoặc mở Developer Tools (F12) -> Network tab -> Check "Disable cache"
